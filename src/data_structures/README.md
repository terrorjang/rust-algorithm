# Binary Search Tree (BST)

## Time Complexity In big O notation

| Operation | Average  | Worst case |
| :-------: | :------: | :--------: |
|  Search   | O(log n) |    O(n)    |
|  Insert   | O(log n) |    O(n)    |
|  Delete   | O(log n) |    O(n)    |
